# Deep Learning Lab Experiments

TensorFlow/Keras experiments for the Deep Learning Technique Lab (U23CS7L1).

## Experiments

| # | Topic | Dataset |
|---|-------|---------|
| 1 | Tensor creation and basic operations | - |
| 2 | Single-unit perceptron | Iris |
| 3 | MLP with activation functions and optimizers | Breast Cancer Wisconsin |
| 4 | MLP for 28x28 image classification | Fashion-MNIST |
| 5 | CNN for multi-category JPG classification | TF Flowers |
| 6 | CNN overfitting, underfitting, and regularization | MNIST |
| 7 | CNN architectures (LeNet, AlexNet, VGG) | MNIST |
| 8 | Simple RNN for sentiment classification | IMDB |
| 9 | LSTM for sentiment classification | IMDB |
| 10 | GRU for sentiment classification | IMDB |
| 11 | Denoising autoencoder | MNIST |
| 12 | Generative adversarial network | MNIST |

## Requirements

- Python 3.9+
- TensorFlow
- NumPy
- scikit-learn
- matplotlib
